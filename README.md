# HoloCare

HoloCare es una aplicación diseñada para **calcular riesgos de multimorbilidad** y **apoyar el manejo de pacientes** mediante el uso de las **guías clínicas más actualizadas**.

Esta versión corresponde a una **versión de prueba**, desarrollada para evaluación inicial, retroalimentación de usuarios y validación del flujo de trabajo. Algunas funciones, interfaces o recomendaciones pueden cambiar en futuras versiones.

---

## Características principales

- Cálculo de riesgos de multimorbilidad
- Apoyo al manejo y seguimiento de pacientes
- Uso de guías clínicas recientes como base para la toma de decisiones
- Interfaz simple para facilitar su uso en entorno clínico

---

## Descarga

Puedes descargar el instalador más reciente desde la sección de **Releases** de este repositorio.

Archivo esperado:

- `HoloCareSetup.exe`

---

## Instrucciones de instalación

1. Ve a la sección de **Releases** del repositorio.
2. Descarga el archivo **`HoloCareSetup.exe`**.
3. Haz doble clic sobre el instalador para iniciar la instalación.
4. Sigue las instrucciones del asistente de instalación.
5. Una vez completada la instalación, podrás abrir HoloCare desde el menú de inicio o desde el acceso directo creado durante la instalación.

---

## Nota de seguridad

El instalador se distribuye firmado digitalmente. Aun así, dependiendo de la configuración de seguridad de Windows o del antivirus, puede aparecer una advertencia al descargar o ejecutar el archivo.

Si esto ocurre:

- Verifica que el archivo proviene de este repositorio oficial
- Asegúrate de descargarlo únicamente desde la sección de **Releases**
- Revisa la firma digital del instalador en las propiedades del archivo, si lo deseas

---
