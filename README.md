<h1 align="center">HoloCare</h1>

HoloCare es una aplicación diseñada para **calcular riesgos de multimorbilidad** y **apoyar el manejo de pacientes** mediante el uso de las **guías clínicas más actualizadas**.

Esta versión corresponde a una **versión de prueba**, desarrollada para evaluación inicial, retroalimentación de usuarios y validación del flujo de trabajo. Algunas funciones, interfaces o recomendaciones pueden cambiar en futuras versiones.

---

## Características principales

- Cálculo de riesgos de multimorbilidad
- Apoyo al manejo y seguimiento de pacientes
- Uso de guías clínicas recientes como base para la toma de decisiones
- Interfaz simple para facilitar su uso en entorno clínico

---

## Descarga

Puedes descargar el instalador más reciente desde la sección de **Releases** de este repositorio.

Archivo esperado:

- `HoloCareSetup.exe`

---

## Instrucciones de instalación

1. Ve a la sección de **Releases** de este repositorioen la parte derecha, haz click en **Releases** y selecciona la versión más reciente.
2. Descarga el archivo **`HoloCareSetup.exe`** en la sección **Assets**.
3. Al descargarlo, es posible que Windows SmartScreen o tu antivirus muestren una advertencia de seguridad. Esto puede ocurrir aunque el instalador sea legítimo, especialmente en versiones nuevas distribuidas fuera de Microsoft Store.
4. Verifica que el archivo proviene de este repositorio oficial y que corresponde a **`HoloCareSetup.exe`**.
5. Haz doble clic sobre el instalador para iniciar la instalación.
6. Si Windows muestra una advertencia al ejecutar el archivo, selecciona **Más información** y luego **Ejecutar de todas formas**, únicamente después de confirmar que el archivo fue descargado desde la sección oficial de **Releases** de este repositorio.
7. Sigue las instrucciones del asistente de instalación.
8. Una vez completada la instalación, podrás abrir HoloCare desde el menú de inicio o desde el acceso directo creado durante la instalación.

---

## Nota de seguridad

El instalador de HoloCare se distribuye **firmado digitalmente**. Sin embargo, debido a que se trata de una aplicación nueva distribuida fuera de Microsoft Store, algunos sistemas de seguridad de Windows o ciertos antivirus pueden mostrar advertencias al descargar o ejecutar el archivo.

Si esto ocurre:

- Asegúrate de descargar el instalador únicamente desde la sección oficial de **Releases** de este repositorio.
- Verifica que el archivo descargado sea **`HoloCareSetup.exe`**.
- Si deseas, revisa la firma digital del archivo desde **Propiedades > Firmas digitales** en Windows. La firma debe estar a nombre de "Miguel Alejandro Contreras Anes".
- Si SmartScreen muestra una advertencia, puedes continuar seleccionando **Más información** y luego **Ejecutar de todas formas**, siempre que hayas confirmado que el archivo proviene de esta fuente oficial.
- Si tu antivirus bloquea la descarga o ejecución, puede ser necesario autorizar el archivo manualmente o restaurarlo desde la cuarentena, siempre verificando primero que corresponde al instalador oficial de HoloCare.

Estas advertencias no indican que el archivo sea malicioso; se deben a mecanismos de reputación de software aplicados a aplicaciones nuevas.
